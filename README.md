# kanban-task-manager
A Kanban Task Manager built with HTML, CSS &amp; JavaScript  managed with Jira and Confluence
## Live Features
- Add tasks with priority (High, Medium, Low) and category
- Drag and drop tasks between columns
- Move tasks with buttons — Start, Done, Redo
- Delete tasks
- Tasks auto-save in browser

## Tech Stack
- HTML
- CSS
- JavaScript

## Project Management
This project was fully managed using Agile methodology:
- Jira — task tracking and Kanban board
- Confluence — project documentation
- Kanban framework — To Do, In Progress, Done
